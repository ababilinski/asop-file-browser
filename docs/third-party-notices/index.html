Installing safely

Use Managed Copy or your own compatible installation.

In Settings → Tools, choose Managed Copy, then Download. The app downloads the checksum-pinned scrcpy 4.1 archive from its official GitHub release and keeps it in Application Support. No Homebrew, Terminal, or administrator password is required.

Information for people distributing a custom build

The standard app bundle does not contain ADB or scrcpy. A custom build that adds Platform-Tools must include the matching complete notice and unmodified source.properties. A build that redistributes scrcpy must include the licenses, notices, dependent libraries, and any source or relinking materials required by that exact build.

A generic Apache license file does not replace a package’s complete notices. Do not copy an arbitrary Homebrew executable into a release.
