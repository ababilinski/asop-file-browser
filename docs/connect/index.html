What do you want to set up?

File Transfer Mode

Plug in. Start browsing.

The simplest way to move files.

  1. 1

    Plug in your phone.

    Use a USB cable that can transfer data. Unlock the phone.

  2. 2

    Choose File transfer.

    Open the USB notification on your phone, then choose File transfer / Android Auto.

  3. 3

    Scan your phone.

    In the Mac app, choose File Transfer Mode, then Scan Phone. Allow access if macOS asks.

  4. 4

    Open File Transfer.

    Your files are ready to browse, preview, and move.

Developer Options

Set up once. Choose USB or Wi-Fi.

For Phone Tools and wireless access.

Enable Developer Options.

  1. 1

    Open Settings.

  2. 2

    Tap About phone.

    On some phones, tap Software information next.

  3. 3

    Tap Build number seven times.

    Enter your PIN if Android asks.

  4. 4

    Open Developer options.

    Go back to Settings. It is usually under System.

Then choose a connection

USB debugging or Wi-Fi debugging.

01

USB debugging.

Use the same cable for Phone Tools.

  1. 1

    Turn on USB debugging.

    Open Developer Options, switch on USB debugging, and confirm.

  2. 2

    Connect and approve.

    Plug in the unlocked phone. Tap Allow when it asks about USB debugging.

  3. 3

    Refresh the connection.

    In the Mac app, choose Refresh USB Connection.

02

Wi-Fi debugging.

Pair without leaving a cable connected. Android 11 or later.

  1. 1

    Turn on debugging.

    In Developer Options, turn on Wireless debugging. Use the same Wi-Fi as your Mac.

  2. 2

    Open QR pairing.

    On your Mac, choose Pair with Wi-Fi QR Code. On the phone, tap Pair device with QR code.

  3. 3

    Scan and connect.

    Use your phone to scan the code shown on your Mac. The app connects when pairing finishes.

Quick fixes

Not connected?

See all questions and troubleshooting ›
File Transfer Mode not finding the phone?

Unlock it, choose File transfer, then scan again.

No File transfer option?

Try another cable or USB port. Some cables only charge.

USB debugging not connecting?

Unlock the phone, reconnect the cable, approve the prompt, then refresh the USB connection.

Wi-Fi pairing stuck?

Check that both devices use the same Wi-Fi and keep the QR screen open.
